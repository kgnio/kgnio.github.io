# kgnio.github.io

Personal GitHub Pages repository for hosting a lightweight public website.

This repository is used as a simple static site that can be served through GitHub Pages. It can be extended over time with project links, portfolio sections, notes, or other public content.

---

## Purpose

The goal of this repository is to keep a small and maintainable public website under the `kgnio.github.io` GitHub Pages domain.

It can be used for:

- Personal profile content
- Project showcase pages
- Static documentation pages
- Lightweight portfolio experiments

---

## GitHub Pages

This repository follows the GitHub Pages convention:

```text
username.github.io
```

When GitHub Pages is enabled, the site is published from the selected branch and directory in the repository settings.

For this repository, the expected public URL is:

```text
https://kgnio.github.io
```

---

## Development Workflow

For a simple static site, changes can usually be made directly to HTML, CSS, or JavaScript files.

A typical workflow:

```bash
git checkout -b docs/update-site-content
git add .
git commit -m "docs: update site content"
git push origin docs/update-site-content
```

Then open a pull request and merge it into `main`.
